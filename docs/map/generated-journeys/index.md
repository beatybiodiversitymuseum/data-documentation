This project uses generative AI to create journeys throughout the museum floor. [You can see a demo without AI here as an example](demo/). The idea was to take all of the specimens in the museum, cross-reference them with data from Wikipedia and our [floor map](/map/), then create little custom journeys based on user queries.

Credit for this demo goes to [Ethaniel Chou](https://ca.linkedin.com/in/ethaniel-chou-4b5828232).
