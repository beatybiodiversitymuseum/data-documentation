The museum floor has many items in [drawers](https://explore.beatymuseum.ubc.ca/docs/drawers/), on [labels](https://explore.beatymuseum.ubc.ca/docs/labels/), and hanging in [shadowboxes](https://explore.beatymuseum.ubc.ca/docs/shadowboxes/). 

There is now an [index of all the items on the museum floor](displays/) so that it could be easier to search for any reference material that might be mentioned but not indexed in our database. This just a demo version that we are providing to get feedback from museum staff. To get the data presented here, we scraped the links for [drawers](https://explore.beatymuseum.ubc.ca/docs/drawers/), [labels](https://explore.beatymuseum.ubc.ca/docs/labels/), and [shadowboxes](https://explore.beatymuseum.ubc.ca/docs/shadowboxes/), ran OCR on them, cross-referenced them with our [floor map](/map/), and scraped Wikipedia for common names. The result is an easy, searchable interface for everything on the museum floor.
