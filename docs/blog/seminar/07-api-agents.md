# Data cleaning workflow
In today's seminar, we'll be looking at the Agents table in some detail. The point will be to understand both how the Agent table is structured and can be queried.

To facilitate this, we'll use the perspective of an API. An API is a way for us to get data from web applications without having to go through the user interface. I'll be demonstrating how to read API documentation generally, and also specifically for Specify and the Agents table. I'll also show how this can be used for visualization that can help us increase our trust in our data.
